# Modern Restaurant - 法式餐廳網站樣板

## 樣板介紹
一套為高級法式餐廳設計的 12 頁完整形象網站，採用暖色調大圖背景，營造精緻高雅的用餐氛圍。基於 Bootstrap 5 建構，完整支援 RWD 響應式設計。

## 適合產業
- 法式餐廳 / Fine Dining
- 餐酒館
- 異國料理
- 高級牛排館

## 如何修改品牌名稱
1. 全域搜尋「Pantry & Plates」或「Modern Restaurant」
2. 替換為您的品牌名稱
3. 修改每個頁面的 title 和 meta description

## 如何替換圖片
- 所有圖片使用 Unsplash 可商用圖片

## 圖片來源注意事項
本樣板使用的 Unsplash 圖片僅作為設計示意，正式上線前請替換為客戶自有圖片或取得 Unsplash 授權。
