# 極速汽車美容中心 - 網站模板

## 模板介紹
一套為 極速汽車美容中心 設計的 12 頁完整形象網站，基于 Bootstrap 5 建構，完整支持 RWD 響應式設計。

## 適合產業
- 極速汽車美容中心 相關產業

## 文件結構
`
template-T024-auto-detailing/
+-- index.html ~ contact.html (12頁)
+-- template.json / tags.json / prompt.md / README.md / images-sources.md
+-- assets/
    +-- css/style.css + responsive.css
    +-- js/main.js
    +-- img/preview.jpg
`

## 如何修改品牌名稱
全域搜索品牌名稱替換為您的內容。

## 如何替換圖片
所有圖片使用 Unsplash 可商用圖片，請至 unsplash.com 搜索替換。

## 如何修改 Line/IG/地圖
搜索 line.me、instagram.com 和 Google Maps iframe 替換。

## 如何修改 SEO
每個 HTML 頁面的 title、meta description、og:title、og:description。

## 圖片來源注意事項
本模板 Unsplash 圖片僅為設計示意，正式上線前請替換。
