# Dental Clinic - 牙醫診所網站樣板

## 樣板介紹
一套為牙醫診所設計的 12 頁完整形象網站，採用淺藍潔白色系，營造安心專業的醫療氛圍。基於 Bootstrap 5 建構。

## 適合產業
- 牙醫診所 / 植牙中心 / 矯正專科

## 檔案結構
```
template-T011-dental-clinic/
├── index.html / about.html / services.html / service-detail.html
├── portfolio.html / reviews.html / faq.html / booking.html
├── process.html / blog.html / blog-detail.html / contact.html
├── template.json / tags.json / prompt.md / README.md / images-sources.md
└── assets/
    ├── css/style.css + responsive.css
    ├── js/main.js
    └── img/preview.jpg
```

## 如何修改品牌名稱
全域搜尋「皓亮牙醫診所」或「Dr. Bright Dental」替換為您的品牌。

## 如何替換圖片
所有圖片使用 Unsplash 可商用圖片，可至 unsplash.com 搜尋替換。

## 如何修改 Line/IG/地圖
搜尋 `line.me`、`instagram.com` 和 Google Maps iframe 替換。

## 如何修改 SEO
每個 HTML 頁面的 title、meta description、og:title、og:description。

## 圖片來源注意事項
本樣板 Unsplash 圖片僅為設計示意，正式上線前請替換。
